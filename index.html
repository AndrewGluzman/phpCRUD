Routes

All HTTP methods are supported. You can use http or https for your requests.

  • https://toysphp.000webhostapp.com/prods/list.php (GET)- returns all entries of toys
    • Client may limit number of entries by seting it in perPage and page queries after /list.php/?perPage=5&page=1. e.g 5 entries: https://toysphp.000webhostapp.com/prods/list.php/?perPage=5&page=1this way client may get 10 entries on each page.
  • https://toysphp.000webhostapp.com/prods/list.php/?search="your query" (GET)- Gives an option to search specific toy by name and description
  • https://toysphp.000webhostapp.com/prods/list.php/?catName="your query" (GET)- Gives an option to look for toys in specific toy category. There is three options girls, boys and sport.
  • https://toysphp.000webhostapp.com/prods/list.php/?min="your query"&max="your query" (GET)- Gives an option to look for toys with specific toy price range. client must give parameters to min and max properties.
  • In order to add new toy client need send (POST) request to main adress login required https://toysphp.000webhostapp.com/prods/php_proj.php as following example { "name": "smartphone", "info": "apple", "category": "girls", "img_url": "https://images.pexels.com/photos/187329/pexels-photo-187329.jpeg", "price": 900 } Authotication token needed in order to fulfill addition request. Client must include the authotication token in header inside the request.
  • In order to update existing toy client need send (PUT) request to main adress providing the id of the item you need to update https://toysphp.000webhostapp.com/prods/php_proj.php/?id="youritemid" as following with the property changed: example { "name": "tv", "info": "sony", "category": "girls", "img_url": "https://images.pexels.com/photos/187329/pexels-photo-187329.jpeg", "price": 900 } Authotication token needed in order to fulfill update request. Client must include the authotication token in header inside the request.
  • For Deletition of a toy client must provide id of the item as follows https://toysphp.000webhostapp.com/prods/php_proj.php/?id="youritemid" Authotication token needed in order to fulfill update request. Client must include the authotication token in header inside the request.
